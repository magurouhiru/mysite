フレックスボックス

  要素の種類の認識。
  CSS のdisplay: flex で使用可能。
  複数の子要素を並べたいときに便利。
  主軸を設定することで、子要素を縦とか横とか好きな方向に並べることができる。
  また、主軸と交差軸(主軸と垂直方向)の子要素の長さはデフォルトで統一される。

  tips:flex-direction: column;を使うことで縦方向にも枠を使える範囲でフルに使える。あとは横の長さをscreen とかすればいい感じになることも。
  実際このサイトでも使用している。

  ※注意flex 属性は同時にいくつか設定できるが、設定する数と単位によって振る舞いが変わる。
  詳細はmdn 先生に聞いてね。
